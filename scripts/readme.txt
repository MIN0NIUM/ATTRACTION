// 概要 ------------------------------------------ //

作成したpsdファイルについて、レイヤカンプを自動で作成します。
このスクリプトを使用するにはドキュメントに記載のあるレイヤ構造に沿っている必要があります。

ドキュメント：https://vivid-winter-008.notion.site/a2db4ffe827341ae984e09cad9c99614

このスクリプトはAdobe Photoshop バージョン : 25.5.0 で作成されています。


// スクリプトの登録 ------------------------------ //

このスクリプトを使用するためには以下の手順を踏む必要があります。

①スクリプトを保存する
まず、Photoshopのスクリプトフォルダにスクリプトファイル（拡張子が.jsx）を保存します。この場所は、通常、以下の場所にあります。
・Windows: C:\Program Files\Adobe\Adobe Photoshop [バージョン]\Presets\Scripts
・macOS: /Applications/Adobe Photoshop [バージョン]/Presets/Scripts

②Photoshopを再起動する
スクリプトを保存した後、Photoshopを再起動します。これにより、Photoshopがスクリプトを認識し、メニューに追加されます。

③スクリプトを実行する
Photoshopを開いた後、メニューバーの「ファイル」メニューに移動します。そこに、保存したスクリプトが表示されます。クリックすると、スクリプトが実行されます。
